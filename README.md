# NPMCommands

##

All of these scripts add the following submodules by GitHub@jauntyjocularjay:
- kwikgit in `./git`

## static.sh

Static creates a boilerplate for a basic, static webpage.

```
bash static.sh
```

## typescript.sh

Creates a boilerplate for a basic typescript with:
- ES6 syntax
- Jest testing
- babel

```
bash typescript.sh
```
